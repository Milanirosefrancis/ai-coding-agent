# AI Coding Agent

An autonomous AI coding assistant built with:

- Ollama (DeepSeek Coder)
- Supabase Memory
- Chroma Vector Memory
- Multi-Agent Architecture
- Automatic Tool Execution

## Features

- Project analysis
- File reading/writing
- Code execution
- Persistent memory
- Semantic vector memory
- Planner / Coder / Debugger agents

## Architecture

User
↓
CLI Interface
↓
Core Agent
↓
Planner Agent
↓
Coder Agent
↓
Debugger Agent
↓
Memory Layer
↓
Vector Database
↓
Tools Layer
↓
Ollama Model
